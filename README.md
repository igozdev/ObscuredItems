# ObscuredItems
itch.io Page: https://igoz.itch.io/obscured-items
<br><br>
A collection of types made to be hidden from memory scanners such as Cheat Engine for use in the Unity game engine.
## Supported Types
- bool
- byte
- sbyte
- char
- short
- ushort
- int
- uint
- long
- ulong
- double
- float
